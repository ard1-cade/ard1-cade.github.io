# Web Buku Tamu Digital

Web aplikasi buku tamu digital berbasis PHP dan MySQL yang digunakan untuk mempermudah pendataan tamu secara digital dan modern.

## Fitur Utama
- Login Admin
- Input Data Tamu
- Open Kamera Saat Pengisian Buku Tamu
- Penyimpanan Data Pengunjung
- Dashboard Admin
- Rekap Data Kunjungan

## Teknologi Yang Digunakan
- PHP
- MySQL
- HTML
- CSS
- Bootstrap
- JavaScript

## Tujuan Project
Project ini dibuat sebagai inovasi digitalisasi buku tamu dengan memadukan fitur kamera sebagai bukti kehadiran pengunjung.
